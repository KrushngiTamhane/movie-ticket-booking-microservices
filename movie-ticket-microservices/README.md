# Movie Ticket Booking Microservices Project

## Services Included
- Eureka Server
- API Gateway
- Movie Service
- Theater Service
- Booking Service
- User Service
- Angular Frontend

## Tech Stack
- Java 17
- Spring Boot
- Spring Cloud
- Eureka Discovery
- Spring Cloud Gateway
- Angular

## Run Order
1. eureka-server
2. api-gateway
3. movie-service
4. theater-service
5. booking-service
6. user-service
7. Angular frontend
